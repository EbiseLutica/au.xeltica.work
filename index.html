Ebing Us!? MODサーバー

Ebing Us!? MODサーバー（通称エビアス鯖）は、普段のAmong Usと違い、Xelticaがセルフ・ホスティングしているAmong Usサーバーです。

自作プラグインで新しい遊びを提供できるという特徴があり、私の実験的機能追加で溢れていく予定です。

簡単なパッチ当てだけで参加でき、通常のAmong Usと同様に、部屋を作ってみんなで遊べます！

独自機能

  • 第三陣営「てるてる」 ベータ版
    • クルーの中からランダムに1人選ばれ、名前欄とチャットで通知される（他の人には見えない）
    • 勝利条件は、「吊られること」
      • 噛まれたり、生存したりしてはいけない
    • よって、てるてるは素性を隠しながら、インポスターを装う必要がある
    • 軽率に吊りづらくなる
開発中の独自機能を見てみる？
  • 第三陣営「恋人」 開発中
    • プレイヤーの中からランダムに2人が選ばれ「恋人」となる
    • ゲーム開始時に自分が恋人ロールであり、相方が誰かが名前欄・チャットに表示（他の人には見えない）
    • 勝利条件は、2人揃って最後まで生き残ること
    • 片方
    • クルー×クルー、 クルー×インポスター、 インポスター×インポスター のカップルがありえる
    • インポスターラインか恋人ラインかの見極めが発生する
    • クルー×インポスターの場合、相方インポスターから恋人を守る必要あり
  • 新役職「マッドメイト（狂人）」、「スパイ」 開発中
    • クルーの中からランダムに選ばれる。
    • クルーとして振る舞うが、勝利条件は「インポスターの勝利」である。
    • スパイはマッドメイトと違い、インポスターが誰か知ることができる。
    • クルーからインポスターを庇う
  • 新役職「猫又」 開発中
    • クルーの中からランダムに選ばれる。
    • インポスターに殺された場合、インポスターも死ぬ
    • 吊られた場合、誰かもランダムに1人吊られる
  • ゲームモード「鬼ごっこ」 開発中
    • ゲーム開始時にインポスターの名前が全員に赤く見える
    • 常にコミュサボ（=情報が見られない）
    • 会議が常に0
    • クルー視野MAX
    • インポ視野0.25

導入方法

参加するためには、お持ちのAmong Usにパッチを当てる必要があります。

「ちょっと難しそう…。」
って思うでしょ？意外と簡単です。

以下にデバイスごとの導入手法を解説しています。自分の所有しているデバイスの欄を御覧ください。

この手順は一度実行するだけでOKです。次回以降は「オンライン」選択後、右下の地球儀から「EbingUs」を選びます。

Windows

  1. EbingUs!? パッチ当てるヤーツをダウンロードします。
  2. ZIPを展開し、EbingUsInstaller.exeを実行します。
  3. SmartScreen などと出た場合も気にせず続行してください。
  4. 画面の指示に従って操作します。
  5. Among Usを立ち上げ、地球儀アイコンから「EbingUs」を選択
  6. 導入完了です。次以降はこれを選択するだけでエビアス鯖に入れます！
古い手順を見る
  1. このJSONファイルをダウンロードします。
  2. Windows+Rキーをデスクトップで押して、「ファイル名を指定して実行」アプリを起動。
  3. %APPDATA%/../LocalLow/Innersloth/Among Usと入力し、OKを押して実行。
  4. 出てきたフォルダに、先ほどダウンロードしたファイルをコピー（上書きしてOK）
  5. Among Usを立ち上げ、地球儀アイコンから「EbingUs」を選択
  6. 導入完了です。次以降はこれを選択するだけでエビアス鯖に入れます！

iOS / Android / macOS(Appleシリコン)

  1. Among Usを起動して、[オンライン]を選択した状態で、このページへ戻ってきてください。
  2. このリンクをクリックする か、以下のQRコードを読み込みます。
  3. 一見何も変わってませんね？右下の地球儀をタップしましょう。EbingUsが追加されているはずなので選択します。
  4. 導入完了です。次以降はこれを選択するだけでエビアス鯖に入れます！
